# 교회 무료홈페이지 홍보 사이트

작은 교회와 미자립교회를 위한 무료 홈페이지 제작 홍보용 정적 사이트입니다.

## 수정해야 하는 값

- `index.html`의 `https://open.kakao.com/o/REPLACE_ME`를 실제 카카오톡 1:1대화방 주소로 교체하세요.
- `index.html`, `robots.txt`, `sitemap.xml`의 `https://your-domain.example`을 실제 배포 도메인으로 교체하세요.
- 배포 후 네이버 서치어드바이저에 사이트를 등록하고 `sitemap.xml`, `robots.txt`를 제출하세요.

## 네이버 검색 등록 체크

- 사이트 제목은 40자 이내, 설명은 80자 이내를 권장합니다.
- `robots.txt`는 사이트 루트에서 열려야 합니다.
- `sitemap.xml`은 실제 도메인 기준 URL을 담아야 합니다.
- 네이버 검색에서 `site:실제도메인`으로 색인 여부를 확인하세요.
